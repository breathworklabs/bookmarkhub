### 📝 Permissions Explained

**Why we need these permissions:**

• **activeTab** - To access the current X/Twitter tab for bookmark extraction
• **storage** - To save your preferences and extension settings locally
• **tabs** - To open the Markspace app when extraction completes
• **scripting** - To inject extraction scripts on X/Twitter pages
• **cookies** - To authenticate with X/Twitter using your existing session
• **host_permissions (x.com, twitter.com)** - To extract bookmarks from X/Twitter
• **host_permissions (localhost)** - To communicate with local Markspace app

All permissions are used solely for the core functionality described above. No data is ever sent to external servers.
