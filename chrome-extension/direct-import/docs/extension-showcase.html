📸 Extension Popup States

The extension popup appears when users click the extension icon in their browser toolbar.

  • State 1: Initial "Ready to import" state with instructions visible
  • State 2: Active importing state with progress bar showing extraction progress
  • State 3: Successful import completion message with total bookmarks count
  • State 4: Compact view after user closes the info box - shows last import timestamp

🐦 Extension Banners on Twitter/X

These banners appear on the Twitter/X bookmarks page (twitter.com/i/bookmarks) when the extension is active.

  • State 1: Initial banner prompts user to start the import process
  • State 2: Progress banner shows real-time bookmark count and API fetch progress
  • State 3: Success banner displays total bookmarks fetched and provides import button

Note: Banners appear as fixed overlays in the top-right corner and use the Twitter dark theme background.
