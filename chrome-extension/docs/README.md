# Chrome Extension Documentation

This directory contains all documentation for the Markspace Chrome Extension.

## Directory Structure

```
docs/
└── store-submission/          # Chrome Web Store submission materials
    ├── STORE_LISTING.md       # Store listing copy (title, description, etc.)
    ├── PRIVACY_POLICY.md      # Complete privacy policy
    ├── SUBMISSION_CHECKLIST.md # Step-by-step submission guide
    └── screenshots/           # Store screenshots and promotional images
        └── README.md          # Screenshot requirements and tips

tools/
└── generate-markspace-icons.html  # Icon generator tool
```

## Getting Started with Submission

1. **Read the checklist**: Start with [SUBMISSION_CHECKLIST.md](./store-submission/SUBMISSION_CHECKLIST.md)
2. **Generate icons**: Use the [icon generator tool](../tools/generate-markspace-icons.html)
3. **Take screenshots**: Follow the [screenshots guide](./store-submission/screenshots/README.md)
4. **Host privacy policy**: Upload [PRIVACY_POLICY.md](./store-submission/PRIVACY_POLICY.md) to your website
5. **Package extension**: Follow instructions in the checklist

## Important Notes

- All submission materials are ready to use
- Update manifest.json with Markspace branding before submitting
- Test the extension thoroughly before packaging
- Host the privacy policy online (required by Chrome Web Store)
